## Crud básico de inserção e listagem de alunos em Python
#Projeto básico para desenvolvimento de habilidades fundamentais em Python
> Uma evolução do [Extrato de Gastos](https://github.com/AugustoEng?tab=repositories)

Armazena os dados em um arquivo txt, "alunos.txt"

O script possui quatro opções dispostas no Menu:
- Listar Alunos: Lista todos os alunos presentes no banco de dados e suas respectivas informações (Nome, matrícula, idade e curso)
- Adicionar Aluno: Adciona um novo aluno com Matrícula automática
- Remover Aluno: Remove um aluno do banco junto de todos os seus dados
- Mostrar Média Etária: Mostra a idade média de todos os alunos
- Sair: Fecha o programa
